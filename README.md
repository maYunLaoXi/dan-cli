## 针对官网多页面前端开发的脚手架

基于boostrap4, webpack4, jquery, i18n

### usage
```
npm install dan-cli -g
// 或
yarn global add dan-cli
```

```
dan --version
```

快速创建名为dan-demo的项目
```
dan create dan-demo
```

或
init
```
dan init 
```

文档: [github.com/maYunLaoXi/official-web-starter](https://github.com/maYunLaoXi/official-web-starter)
